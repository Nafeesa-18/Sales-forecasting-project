# Sales Forecasting Project

## Overview
This project predicts future sales using Machine Learning and Linear Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Features
- Sales prediction
- Future forecasting
- Data visualization

## Algorithm
Linear Regression

## Project Workflow
1. Load dataset
2. Train model
3. Predict sales
4. Visualize data

## Output
The model predicts future sales based on historical sales data.
